# Calculadora

Projeto de calculadora desenvolvido com HTML, CSS e JavaScript com foco na prática de lógica de programação, manipulação do DOM e eventos.

A aplicação realiza operações matemáticas básicas de forma dinâmica através de uma interface interativa e responsiva, permitindo uma melhor experiência tanto em computadores quanto em dispositivos móveis.

## Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão
* Limpar visor
* Interface responsiva
* Interação dinâmica com botões

## Linguagens

* HTML5
* CSS3
* JavaScript
